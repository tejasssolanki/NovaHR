# NovaHR - Employee Payroll Management System

## Description
NovaHR is a Java-based CLI Employee Payroll Management System developed using Object-Oriented Programming concepts.

## Features
- Add Employee
- View Employees
- Search Employee
- Delete Employee
- Generate Salary Slip
- Save Employee Data to CSV

## Technologies Used
- Java
- Maven
- IntelliJ IDEA
- Git
- GitHub

## Concepts Used
- Classes & Objects
- Encapsulation
- ArrayList
- File Handling (CSV)
- OOP
